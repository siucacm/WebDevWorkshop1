## HTML:

What is HTML?


HTML is a markup language used in developing web pages.


The building blocks of HTML web pages are called "elements", which are represented by tags.



Common elements:


#### p: Paragraphs


#### h(1-6): Headings

#### a: Links


#### img: Images


#### button: Button


#### ul/li: Unordered/Ordered lists


#### div: Division in a web page
